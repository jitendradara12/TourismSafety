# Architecture Decision Records (ADRs)

This directory stores records of architectural decisions for the SIH Safety Platform.

- Use one ADR per decision.
- Keep ADRs short, specific, and versioned.
- Status values: Proposed, Accepted, Superseded, Deprecated.

Suggested numbering: ADR-0001, ADR-0002, ...
